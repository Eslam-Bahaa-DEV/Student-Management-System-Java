# Student Management System (Java)

## 📌 Description
A console-based Student Management System built using Java.  
The system allows adding, updating, searching, and displaying student records with validation checks.

The project demonstrates Object-Oriented Programming (OOP) concepts such as inheritance, abstraction, and encapsulation.

---

## 🚀 Features
- Add new students
- Prevent duplicate IDs
- Validate age and grade inputs
- Display all students
- Search student by ID
- Update student information (name, ID, age, grade)

---

## 🧠 Concepts Used
- Object-Oriented Programming (OOP)
  - Inheritance
  - Abstraction
  - Encapsulation
- Arrays
- Input validation
- Menu-driven program
